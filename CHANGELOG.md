# Changelog (更新日誌)

本專案的所有顯著變更都將記錄在此文件中。

## [1.1.0] - 2026-01-08

### 🚀 新增功能 (Added)
- **動態槽位系統 (Dynamic Slots)**：
  - 移除固定的 4/8/12 槽位限制。
  - 圖片去文字工作台現在會根據匯入的圖片數量自動擴展，支援無限量批次處理。
- **一鍵全頁導入**：
  - 在 PDF 解析頁面點擊「一鍵導入去字」時，會自動為所有頁面建立工作槽位並導入。
- **Gemini 3 模型整合**：
  - **OCR 辨識**：升級至 `gemini-3-flash-preview`，大幅提升文字辨識速度與精準度。
  - **圖片去字**：升級至 `gemini-3-pro-image-preview`，提供業界領先的圖像修復與背景填補效果。
- **安全 API Key 管理**：
  - 新增「設定」按鈕與專屬設定視窗。
  - 支援將 Google Gemini API Key 安全儲存於瀏覽器 `localStorage`。
  - 應用程式啟動時自動檢查 Key 狀態，若未設定則提示用戶。
- **智慧速率控制 (Smart Rate Limiting)**：
  - 新增「API 模式」切換功能 (免費版/付費版)。
  - **免費版**：強制執行請求間隔延遲，確保符合 15 RPM 限制。
  - **付費版**：啟用多線程平行處理，大幅提升批次作業速度。
  - 設定視窗內建 Google AI Studio 綁卡連結，方便用戶升級。

### 🐛 錯誤修正與優化 (Fixed & Improved)
- **安全性優化**：移除了程式碼中硬編碼的 API Key，防止憑證外洩。
- **介面優化**：移除了不再需要的「槽位數量選擇」下拉選單，簡化操作介面。
- **流程優化**：改進了批次處理的佇列邏輯，確保大量圖片處理時的穩定性。

## [1.0.0] - 2026-01-08

### 🎉 首次發布 (Initial Release)
- **單頁應用程式整合**：將原始的 PDF 解析工具與 PPTX 合併工具整合成單一 `index.html`。
- **PDF 解析功能**：
  - 支援 PDF 拖放上傳與預覽。
  - 頁面轉圖片高解析度渲染。
  - 初步支援 AI OCR 轉 PPTX 視覺還原。
- **圖片去字工作台**：
  - 支援單張或批次圖片 AI 去字。
  - 提供 4/8/12 固定槽位模式。
- **PPTX 合併工具**：
  - 支援文字檔 (Text) 與背景檔 (Background) 的 XML 層級合併。
